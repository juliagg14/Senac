Índice

O que é Índice?

O índice é um indicador que serve para analisar como anda o desempenho das movimentações da economia.
Os índices também são utilizados na Bolsa de Valores medindo o desempenho de um conjunto de ações.
Quais são os índices utilizados na bolsa de valores? Entender quais são os índices utilizados na bolsa de valores é essencial para as estratégias de um investidor.

Sumário ou índice?

O que é um sumário e o que é um índice? É muito habitual os editores chamarem índice à lista de assuntos que uma publicação trata e que é inserida no início, como deve ser, ou no fim do volume.
